How to Play Online:

  1. Go to this website:
    Tutoials Point - Unix Terminal Online
    . This loads a unix terminal.
  2. wget the latest linux build: 
    wget https://github.com/nedearb/Underneath/raw/master/Underneath_Optim_Term_Linux
  3. Allow to be executed: 
    chmod +x Underneath_Term_Linux
  4. Run the program! 
    ./Underneath_Term_Linux
